<objective>
Wire Steam installation detection and app ID resolution into the Proton launch pipeline.

Purpose: With FindSteamInstall() from Plan 01, this plan connects the detection to the actual Proton environment variables. SteamGameId gets set to the non-Steam shortcut app ID (enabling Gamescope window tracking across D3D recreation), and STEAM_COMPAT_CLIENT_INSTALL_PATH gets set to the detected Steam root (enabling Proton to find steamclient.so). This directly addresses the controller input loss during ServerTravel.

Output: Updated proton_env.go, pipeline_linux.go with new step, LaunchState/LaunchConfig with new fields, exported FindCluckersAppID, and updated tests.
</objective>

    3. In `pipeline_linux.go`, add a new pipeline step `stepResolveSteamIntegration`:
       ```go
       func stepResolveSteamIntegration(_ context.Context, state *LaunchState) error {
           steamDir := wine.FindSteamInstall()
           if steamDir == "" {
               ui.Verbose("Steam installation not found, controller tracking may be limited", state.Config.Verbose)
               return nil // Non-fatal
           }
           state.SteamInstallPath = steamDir
           ui.Verbose(fmt.Sprintf("Steam: %s", steamDir), state.Config.Verbose)

           // Resolve non-Steam game app ID from shortcuts.vdf.
           pattern := filepath.Join(steamDir, "userdata", "*", "config", "shortcuts.vdf")
           matches, _ := filepath.Glob(pattern)
           for _, shortcutsPath := range matches {
               data, err := os.ReadFile(shortcutsPath)
               if err != nil {
                   continue
               }
               if appID := FindCluckersAppID(data); appID != 0 {
                   state.SteamGameId = fmt.Sprintf("%d", appID)
                   ui.Verbose(fmt.Sprintf("Steam shortcut app ID: %s", state.SteamGameId), state.Config.Verbose)
                   return nil
               }
           }

           ui.Verbose("Cluckers shortcut not found in Steam, using default game ID", state.Config.Verbose)
           return nil
       }
       ```
       Add `"path/filepath"` and `"os"` to imports if not already present. Add `"fmt"` if needed.
