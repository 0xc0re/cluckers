<decisions_made>

- **EVIOCGRAB for device isolation**: Proxy exclusively grabs ALL Steam Input pads so Wine/SDL can only see the proxy's virtual Xbox 360 pad. This prevents event competition and multiple-gamepad confusion.
- **Defined local eviocgrab constant (0x40044590)**: `golang.org/x/sys/unix` doesn't export `EVIOCGRAB`. Used `_IOW('E', 0x90, int)` = `0x40044590`.
- **FindAllSteamInputPads**: Grabs ALL matching VID/PID devices (not just first) because Steam creates multiple virtual pads (pad 0, pad 1).
- **Non-fatal proxy**: stepStartInputProxy logs warning and continues if proxy fails — game works without proxy but may have controller drops during ServerTravel.
- **Wine winebus registry**: DisableHidraw=1 + Enable SDL=1 — confirmed patched correctly on Deck (`~/.cluckers/compatdata/pfx/system.reg`).
