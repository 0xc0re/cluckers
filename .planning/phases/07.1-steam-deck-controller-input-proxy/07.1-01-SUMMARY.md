## Decisions Made
- Used raw `unix.Syscall(SYS_IOCTL, ...)` for struct-passing ioctls and `unix.IoctlSetInt` for simple ioctls -- kenshaw/evdev's UserInput has incomplete absolute axis support
- Button/axis constants use named Go constants (btnA, absX) rather than inline hex values for readability and package-wide reuse
- Y-axis inversion (invertY) placed in uinput.go alongside axis definitions for co-location with related types
- kenshaw/evdev used for device detection (OpenFile + ID()) but not for uinput creation
